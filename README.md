## Python 코딩 규칙

1. 들여쓰기 : 공백 4칸을 권장

   한 줄은 최대 79자 까지 작성 권장

   클래스 내의 메서드 정의는 1줄씩 띄어 씀

```python
for x in array:
    if x < pivot:
        less.append(x)
    else:
        greator.append(x)
```

: 콜론은 코드 블록의 시작을 의미, 블록이 끝날 때 까지 블록 안에 코드는 모두 같은 크기만큼 들여써야 함

2. 표현식, 문장 : 공백 사용 지양

   대괄호 [ ], 소괄호 ( ) 안에

   쉼표 (,), 쌍점 (:)과 쌍반점 (;) 앞

3. `#` 주석, 또는 코드를 지우지않고 실행만 되지 않도록 남겨두고 싶을 때 활용
