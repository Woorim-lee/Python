## 함수의 구조

함수를 사용하기 위해서는.. 1) 함수 정의(인자) 하고, 2) 함수 호출(인자도 함께 호출)해서 사용할 수 있음!

* 함수 정의

```python
def 함수명([인자1, 인자2, 인자3, ... , 인자n]):
  <코드블록>
  [return <반환값>]
```

맨 마지막줄에 반환값이 있으면 return <반환값> 을 입력하고 없으면 아무것도 입력하지 않음

<br>

* 함수 호출

```python
함수명([인자1, 인자2, 인자3, ... , 인자n])
```
